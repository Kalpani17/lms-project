📘 LMS Landing Page

A responsive front-end Learning Management System (LMS) landing page developed using HTML, CSS, JavaScript, and Bootstrap. This project demonstrates modern UI design, responsive layout techniques, and interactive features for an educational platform.

🚀 Features:
  🎨 Responsive Design
  Built using Bootstrap and media queries to ensure compatibility across desktop, tablet, and mobile devices.
  🌙 Dark Mode Toggle
  Users can switch between light and dark themes, with preferences saved using local storage.
  ⏰ Dynamic Greeting System
  Displays a greeting message (Morning/Afternoon/Evening) based on the user’s current time.
  🧭 Navigation Bar
  Clean and responsive navigation menu with multiple sections (Home, Courses, About, Contact).
  📚 Course Display Section
  Showcases popular courses with images, descriptions, and action buttons.
  💡 Feature Highlights Section
  Displays key LMS features such as course library, discussions, and certification.

🛠️ Technologies Used:
  HTML
  CSS
  JavaScript 
  Bootstrap
